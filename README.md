## Building

- Clone the repo
- Downloading SFML
 - Automatic (Windows only)
   - Run `.download_sfml.bat`
 - Manual
   - Download the latest version of [SFML](https://www.sfml-dev.org/download.php)
     - Extract to `SFML` in the root folder of project
       - The folder structure should look like:
         - `~/main.cpp`
         - `~/SFML/bin/.`
         - `~/SFML/lib/.`
         - ect...
